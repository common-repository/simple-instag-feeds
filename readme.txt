== Description ==

The Simple Instag Feeds is available for those with the Wordpress platform website. It is available for all versions of Wordpress and is easy to install, easy to use and something that brings a slew of awesome benefits to the visitor as well as the website owner.


Features and benefits of the Simple Instag Feeds that you will enjoy include the following:

i)  Show/Hide Fronted view using setting.

ii) Use Shortcode

Easy to use Different page or post by shortcode "[simple_intag_widget_display]".

Use This Code in Page Template <code> <?php echo do_shortcode('[simple_intag_widget_display]'); ?> </code>.

The Simple Instag Feeds can easily be added to your site with all of the above benefits.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first

You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/directory

2. Activate the plugin

3. You will see "Simple Instag Feeds" settings from right side of your wordpress menu.
